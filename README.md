# Demo Movie-website - Frontend (client)

## System requirements
  * Node.js

## Project configuration
  * Declare environment variables in .env-example file and rename it to .env

## Installation
  * npm i

## Project scripts
  * npm start - launches development server
  * npm build - compile app
